### O que é a TAG div

-Div é uma tag que cria um bloco genérico para agrupar elementos. Existem outras tags que podemos usar para agrupar elementos no HTML, porém a div é a mais genérica.

### Outros exemplos:

- div
- nav
- article
- section
- header
